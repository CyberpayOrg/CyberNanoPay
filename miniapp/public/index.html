How the Loop Works
1
You deposit USDT via Top Up — on-chain, one-time per session
2
AI Agent pays APIs via off-chain Ed25519 signature — zero gas, ~1ms per payment
3
TEE aggregates payments and batch-settles to TON chain periodically
4
You approve large payments here — HITL with full control
Agent Signing Keypair (Ed25519)
Setup: Set NANO_AGENT_SEED=<seed> in your agent environment. The agent uses this key to sign all payment authorizations.
